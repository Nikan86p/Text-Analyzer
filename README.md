# Text-Analyzer
This code helps you to analyze your txt file simply!!!
you should use {sample.txt} file and add your text to that or just simply add another txt file that you want and rename the {name} inside the code,plus you should use {requirements.txt} file to install the packages of this code inside your venv.
Then you can start the code and get the results like amount of lines,words, alphabets and etc,plus it gives you the n top words count uses in the txt file.
In conclusion it will give you the main information about that txt file that you want!!!
